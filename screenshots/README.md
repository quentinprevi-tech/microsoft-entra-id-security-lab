# Screenshots

This directory contains anonymized screenshots of the Microsoft Entra ID security lab.

Before publishing screenshots, the following information must be masked:

- personal email addresses
- tenant IDs
- subscription IDs
- application/client IDs
- object IDs
- public IP addresses
- request IDs
- correlation IDs
- generated `.onmicrosoft.com` domains containing personal identifiers
